# Catalog, Booking, and Payments

## Browse catalog

1. Open the wraps catalog.
2. Use filters/search to narrow options.
3. Open an item to view details.

## Book a slot

1. Select the desired service/date.
2. Choose from available slots.
3. Submit booking details.

Slot availability is validated at booking time.

## Checkout and invoices

1. Start checkout from the booking flow.
2. Complete payment using the provided Stripe checkout page.
3. Return to the app and verify invoice/payment status.
